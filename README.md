# Questie War Within Skin

A "The War Within" style skin for the Questie tracker.

![Preview Image of Questie - german](preview.png)

## Features
- Skinned tracker header with full-width background
- Zone headers use the War Within header texture
- Quest title highlight uses a custom texture
- No configuration or slash commands

## Installation
1. Copy the `QuestieWarWithinSkin` folder into your WoW AddOns directory.
2. Make sure `Questie` is enabled: https://github.com/Questie/Questie
3. Reload your UI.
